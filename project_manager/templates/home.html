This website allows you to register and track all the projects running in the sequencing/bioinformatics facilities in iBiMED. You need to have valid credentials to log in. There are two types of credentials: i) Superuser - allows you to create and consult the information; ii) Staff - only allows you to consult the information. All logins are traceable so that all the information will be tracked.

All personal data is encrypted in the database.

The base model of this website is an association of events to its projects. For each project created, you can add successive events, like ‘samples in preparation’, ‘sequencing’, ‘bioinformatics analysis’, ‘report delivery’, etc. You can also add files to each event to support your actions.
